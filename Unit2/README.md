

# 📘 Unit 2 – Supervised Learning using Python

This section contains Python programs demonstrating **Supervised Machine Learning algorithms** such as classification and regression.  
The experiments illustrate how models learn from labeled datasets and make predictions using different algorithms.

The programs use Python libraries like **NumPy, Matplotlib, Scikit-learn, and mglearn** to visualize datasets and train models.

---

## 🎯 Objectives

- Understand the concepts of **classification and regression**
- Implement **K-Nearest Neighbors (KNN)** for classification and regression
- Explore **linear regression models**
- Understand **regularization techniques** such as Ridge and Lasso
- Learn **linear models for classification**

---

## 🛠️ Libraries Used

- 🐍 Python  
- 🔢 NumPy  
- 🐼 Pandas  
- 📊 Matplotlib  
- 🤖 Scikit-learn  
- 📚 mglearn
